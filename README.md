Add-Content README.md "# ERP - Sistema de Gestão para Fábrica 3D

## Backend (Node.js + Express + PostgreSQL)
- Instalar dependências: `cd erp-backend && npm install`
- Configurar `.env` com credenciais do banco
- Executar: `npm run dev`

## Frontend (React + Vite + Tailwind)
- Instalar dependências: `cd erp-frontend && npm install`
- Executar: `npm run dev`

## Funcionalidades
- Gestão de clientes, produtos, matérias-primas
- Encomendas com status e pagamento
- Financeiro com transações automáticas
- Exportação de relatórios (PDF, Excel, CSV)
- Dashboard com gráficos e alertas"
